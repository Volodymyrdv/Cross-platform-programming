Завдання

Створити батьківський абстрактий клас ВоднийТранспорт, який містить поля для назви транспорту, швидкості та місткості, а також методи для отримання швидкості та виведення інформації про транспорт. На основі цього класу реалізувати два підкласи: Катер, який має додаткове поле для типу двигуна, та Лодка, яка містить поле, що вказує на наявність веслування. Далі потрібно реалізувати функціональність для знаходження n найшвидших транспортів залежно від їх швидкості.
